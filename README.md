# try-heptagon

A Js_of_ocaml powered web-interface to manipulate the Heptagon compiler

## Opam Dependencies

* ocamlbuild
* js_of_ocaml.5.8.2
* js_of_ocaml-ppx.5.8.2
* js_of_ocaml-tyxml.5.8.2
* ezjs_ace.0.1.1
* chartjs.0.2.2

## Compilation

- `git submodule init && git submodule update`
- `cd heptagon && ./configure`
- `make`
